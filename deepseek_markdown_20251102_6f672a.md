# WhatsApp点击追踪器

一个基于GitHub Pages的WhatsApp群链接点击统计工具。

## 功能特点

- 📊 点击次数统计
- 📱 设备信息检测  
- 🌐 来源追踪
- 📈 数据可视化
- 💾 本地数据存储
- 📥 数据导出功能

## 使用方法

1. 将整个项目上传到GitHub
2. 启用GitHub Pages功能
3. 修改 `index.html` 中的WhatsApp群链接
4. 分享你的跟踪链接

## 文件说明

- `index.html` - 主跟踪页面
- `dashboard.html` - 数据查看面板
- `script.js` - 核心逻辑
- `style.css` - 样式文件
- `config.js` - 配置文件

## 注意事项

- 数据存储在浏览器本地，不同设备间不共享
- 清空浏览器数据会丢失统计信息
- 适合小型项目和个人使用
